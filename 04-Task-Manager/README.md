# 🎫 Ticket TM-001 - Slow Computer Investigation

## Overview

Task Manager is a Windows administration tool used to monitor system performance, running processes, startup applications and resource utilization.

This lab demonstrates how to investigate a user complaint regarding poor system performance using Task Manager.

---

# Ticket Information

---

# Reported Issue

User reports that the laptop is running slowly and applications take longer than expected to open.

---

# Objective

Investigate system performance using Task Manager and identify any processes or applications consuming excessive system resources.

---

# Investigation Steps

1. Open Task Manager.
2. Review running processes.
3. Identify high CPU usage.
4. Identify high memory usage.
5. Review disk activity.
6. Review network activity.
7. Examine startup applications.
8. Document findings.
