# Lessons Learned

This lab improved my understanding of Windows System Information and its role in troubleshooting and asset management.

Key lessons learned:

- System Information provides a detailed overview of hardware and software configuration.
- Device manufacturer and model information are important for driver and support requests.
- Processor specifications help determine system capabilities and performance.
- Installed memory helps assess system performance and upgrade opportunities.
- BIOS information is useful when investigating firmware, security and compatibility issues.
- Operating system version information is critical when troubleshooting software and update-related issues.

This lab strengthened my ability to collect and document endpoint information in a professional IT support environment.

No compatibility issues were identified.

The issue would require further application-level troubleshooting.
