# 🖥️ Windows Administration Lab

## Overview

This repository documents my hands-on Windows Administration learning journey as I develop practical skills required for IT Support, Desktop Support, Service Desk and Systems Administration roles.

The purpose of this lab is to gain practical experience using built-in Windows administrative tools while documenting each task as if working in a real IT environment.

---

# Objectives

Throughout this lab I will learn how to:

- Manage Windows devices
- Troubleshoot hardware and software issues
- Configure Windows settings
- Monitor system health
- Manage users
- Investigate system logs
- Understand Windows services
- Perform desktop support tasks
- Improve troubleshooting skills
